# PAN-Number-Validation-Case-Study-python
End-to-end PAN number validation project using Python, regex, and data analysis to identify invalid formats and generate validation insights.

## Project Overview

This project focuses on validating and analyzing Indian PAN (Permanent Account Number) records using Python. The objective was to identify invalid PAN formats, clean inconsistencies, and generate insights through structured data validation and exploratory analysis.
The project simulates a real-world data quality validation task commonly performed in financial and compliance domains.

## Problem Statement
  - Validate PAN numbers based on official format rules and:
  - Identify invalid PAN formats
  - Detect inconsistencies in dataset
  - Perform data cleaning
  - Generate validation summary insights

## Tools & Technologies Used
  - Python
  - Pandas
  - NumPy
  - Regular Expressions (Regex)
  - Jupyter Notebook
  - Microsoft Excel (Final Summary Report)


## Dataset Description
The dataset contains PAN number records requiring validation.
Files Included:
    - PAN Number Validation - Problem Statement.pdf
    - PAN Number Validation Dataset.xlsx (Raw Data)
    - CS_PAN_validation.ipynb (Analysis & Validation Code)
    - CS_PAN VALIDATION REPORT.xlsx (Final Summary Output)

## Key Analysis Performed
  - Identified valid vs invalid PAN numbers
  - Counted format violations
  - Cleaned inconsistent entries
  - Generated summary statistics
  - Created validation report in Excel

## Output & Results
  - Total Records Analyzed: 10000
  - Valid PAN Count: 3193
  - Invalid PAN Count: 5832
  - Missing PAN Count: 975
